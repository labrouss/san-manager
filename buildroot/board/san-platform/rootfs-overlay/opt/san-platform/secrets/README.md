## /opt/san-platform/secrets/

These files are generated by post-build.sh during the Buildroot image build.
They contain PLACEHOLDER values and MUST be changed by the operator after
first boot.

Files:
  db_password.txt    — PostgreSQL password for san_admin (default: san_secret)
  mds_password.txt   — MDS switch admin password (default: CHANGE_ME)
  jwt_secret.txt     — Generated automatically on first boot by write-runtime-env.sh

To change a secret:
  1. SSH into the appliance (or use the VM console)
  2. Edit the relevant file:
       vi /opt/san-platform/secrets/db_password.txt
  3. Restart the platform:
       systemctl restart san-platform

Permissions must remain 600 (root:root only):
  chmod 600 /opt/san-platform/secrets/*.txt
